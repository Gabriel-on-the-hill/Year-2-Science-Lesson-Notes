What is a Variable? 📦

A Variable is a labelled box that holds a number. The game uses it to remember things — like your score!

Where to find it in Scratch:

Click the Variables palette (dark orange) → Click "Make a Variable" → Type a name like Score

Score: 0

The Scratch Blocks:

when 🏴 clicked

set [Score v] to (0)
when this sprite clicked

change [Score v] by (1)

Lives: Counting Down ❤️

Games also track bad things. When a ghost escapes, you lose a life. The variable goes down by using a negative number!

Lives: 3
❤️ ❤️ ❤️

The Scratch Blocks:

when 🏴 clicked

set [Lives v] to (3)
When ghost escapes:
change [Lives v] by (-1)
Key Insight: change by (1) = goes UP. change by (-1) = goes DOWN!

👻 Ghost Catcher:

Lives starts at 3 and goes down when a ghost escapes.

⚽ Penalty Shootout:

Kicks starts at 5 and goes down after each kick attempt.

The Ghost Escape! 💨

IF / THEN: Game Over! 🛑

We need the game to stop automatically when lives reach zero. The IF block checks a condition every time the loop runs.

The Scratch Block:

if <(Lives) = (0)> then

say [Game Over!] for (2) secs

stop [all v]

end
Where does this go? Inside your forever loop, right after change [Lives] by (-1). It checks every time a ghost escapes!

Test the Condition:

Lives = 3
if Lives = 0 ? then
stop all 🛑

👻 Ghost Catcher:

IF Lives = 0, THEN say "Game Over!" and stop all.

⚽ Penalty Shootout:

IF Kicks = 0, THEN say "Full Time!" and show final score.

Sensing: Detecting Contact 📡

How does the ball know it scored? Sensing blocks (light blue) can detect when sprites are touching!

The Scratch Block:

if <touching [Net v] ?> then

change [Goals v] by (1)

play sound [Cheer v] until done

end
Note: In Ghost Catcher we used when this sprite clicked — that's an Event (yellow). Sensing is different — it detects touching automatically!
